# Vinay Portfolio

A personal portfolio website built with Next.js App Router and prepared for Vercel deployment.

## Pages

- Home page
- Projects page
- Contact page

## Local development

```bash
npm install
npm run dev
```

## Deploy

Push this repository to GitHub, then import it into Vercel. Vercel will detect Next.js automatically.

## Live Site

Production URL: https://alphatest-beta.vercel.app

## Contact Form Email

Set `RESEND_API_KEY` in Vercel project environment variables to enable
portfolio contact emails to be sent to `vinayyadav2546@gmail.com`.
